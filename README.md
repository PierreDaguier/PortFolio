# Pierre Portfolio

Modern freelance portfolio rebuilt with React + Vite.

## Tech stack
- React 19
- Vite 5
- Custom CSS (responsive, animation-ready)

## Run locally
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
npm run preview
```

## Included projects
- JCU Weather Prediction: https://jcu-weather-prediction.netlify.app
- CryptoCactus: https://www.cryptocactus.net
